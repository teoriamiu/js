JavaScript: lógica, eventos y comportamiento

Este documento es una clase completa de JavaScript. No enseña diseño ni estructura visual: enseña comportamiento, lógica y tiempo.

Todo el código JavaScript está contenido en este archivo y se ejecuta directamente en el navegador.

1. Variables y tipos de datos

JavaScript trabaja con valores y referencias. El tipo pertenece al valor, no a la variable. 

let numero = 42;
let texto = "Hola mundo";
let activo = true;
let indefinido;
let vacio = null;

2. Funciones

Las funciones describen comportamientos reutilizables. En JavaScript son ciudadanos de primera clase. 

function saludar(nombre) {
  return "Hola " + nombre;
}

3. Condiciones

Las estructuras condicionales permiten tomar decisiones lógicas. 

let edad = 18;

if (edad >= 18) {
  acceso = "Permitido";
} else {
  acceso = "Denegado";
}

4. Bucles

Los bucles permiten repetir procesos de forma controlada. 

for (let i = 1; i <= 5; i++) {
  console.log(i);
}

5. Eventos

JavaScript reacciona a eventos producidos por el entorno (clics, teclas, tiempo, etc.).

6. Tiempo y asincronía

JavaScript no bloquea el flujo principal. Usa un modelo basado en eventos y cola de tareas. 

setTimeout(() => {
  console.log("Esto ocurre después");
}, 2000);

7. Manipulación del DOM

JavaScript puede leer y modificar el DOM, que es el modelo en memoria del documento HTML.
